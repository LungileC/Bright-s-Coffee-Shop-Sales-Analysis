# Bright-s-Coffee-Shop-Sales-Analysis

**A data analytics case study for BrightLearn | Due: March 31, 2026**

This repository contains a comprehensive sales analysis for Bright Coffee Shop, a multi-location coffee business seeking to grow revenue and improve product performance. As a Junior Data Analyst, I extracted actionable insights from 149,116 transactions across 3 store locations to assist the new CEO in strategic decision-making.

Project Objectives 
**Which products generate the most revenue**
**What time of day the store performs best**
**Sales trends across products and time intervals**
**Recommendations for improving sales performance**

Tools 

**Project Planning** MIRO
**Data Processing**  DATABRICKS SQL
**Analysis & Visualization** MS Excel and Powerpoint
**Control GITHUB**
